# 🚛 Arisudan Trucks

A Progressive Web App (PWA) for transport booking and fleet management.

## Features

- Customer Login
- Driver Login
- Admin Dashboard
- Vehicle Booking
- Booking History
- Live Trip Tracking
- Fare Calculator
- Driver Management
- Fleet Management
- Payment Module
- Notifications
- Progressive Web App
- Offline Support
- Firebase Ready

## Technologies

- HTML5
- CSS3
- JavaScript
- Firebase
- Progressive Web App
- LocalStorage
- Service Worker
